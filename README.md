# Obsidian Portal Mapper

Generates GraphViz diagrams based on data stored on [Obsidian Portal](https://obsidianportal.com/).

### Prerequisites

GraphViz

### Installing

```npm install```

### Running

Copy `config.sample.json` to `config.json` and put real values in it

```npm run start```
